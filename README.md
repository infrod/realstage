# Journal de stage CFP — Release

Projet React/Vite prêt pour Vercel.

## Déploiement Vercel

1. Dézipper ce dossier.
2. Envoyer tous les fichiers dans ton dépôt GitHub.
3. Dans Vercel, importer le dépôt.
4. Garder les réglages par défaut :
   - Framework : Vite
   - Build command : npm run build
   - Output directory : dist
5. Cliquer Deploy.

## Test local optionnel

```bash
npm install
npm run dev
```
